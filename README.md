# 🐷 今天吃什么 — 深圳宝安区美食指南

一个可爱的 lulu 猪风格的点餐网页，收录了深圳宝安区 **233 家**高评分餐厅。

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 🍽️ **菜单浏览** | 按菜系/区域筛选，搜索餐厅名 |
| 🎲 **随机选餐** | 限定菜系/区域/价位，一键随机！ |
| ❤️ **收藏/踩雷** | 喜欢的收藏，踩雷的标记后会排除 |
| 📋 **点餐记录** | 自动记录吃过什么，统计次数 |
| 🎨 **lulu猪风格** | 粉嫩可爱 UI，女朋友用了都说好 |

## 🚀 使用

直接访问 GitHub Pages 链接即可：
**https://whitelife0.github.io/baoan-eat-anything/**

所有数据保存在浏览器本地，下次打开不影响。

## 📊 数据

- 233 家宝安区高评分餐厅
- 涵盖 14 大菜系：粤菜、川菜、火锅、日料、韩式、西餐等
- 来自大众点评、小红书等平台
